##音频播放
不管是简单的短促尖声还是交响乐又或是匪帮说唱，Android应用通常需要播放音频。Android中一些能够自动播放音频，例如通知。但是以一旦播了，就不能播第二次。

幸运的是，Android提供了音频录音重放的支持，并且我们会在这章中分析其中的一些选项。
